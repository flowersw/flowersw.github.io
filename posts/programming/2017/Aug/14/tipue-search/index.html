Tipue Search

Posted on 14 August 2017 in Programming

I recently reconstructed my personal website using Pelican, and I went with a pre-made theme called Flex. This theme is a solid one, but lacked search capability out of the box. Fortunately, there is a Pelican plugin called Tipue Search which integrates the Tipue JQuery plugin into Pelican.

Installation into Pelican

It’s easiest to download all of the pelican plugins. I cloned mine at the same level as my pelican repo. You’ll need BeautifulSoup as well.

git clone --recursive https://github.com/getpelican/pelican-plugins
pip install beautifulsoup4

Then, point to the tipue_search plugin in your pelicanconf.py file:

PLUGIN_PATHS = ['path/to/pelican-plugins(can be relative)']
PLUGINS = ['tipue_search']

Installation into your Theme

  1. Download tipue and unpack it.
  2. Copy the tipuesearch/Tipue Search 6.XX/tipuesearch folder to the static directory of whichever theme you are using.

  3. Include the tipuesearch css and search box in your base.html template.

    • In your section, include the tipuesearch css <link rel="stylesheet" href="{{ SITEURL }}/{{ THEME_STATIC_DIR }}/tipuesearch/css/tipuesearch.css">

    • Put the search box form in wherever you want to put it:

  4. Create the search.html page and place it in the templates folder of your theme directory. This is where search results will be displayed, and my version is below.

  5. Don’t forget to add search to your DIRECT_TEMPLATES in your pelicanconf.py

    DIRECT_TEMPLATES = ['index', 'categories', 'authors', 'archives', 'search']

  6. Feel free to customize the way the search box looks. I added rounded edges by setting border-radius: 15px; on the #tipue_search_input element in the tipuesearch.css file.

  7. Enjoy fast, accurate search with Tipue!
